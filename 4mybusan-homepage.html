설정

이 페이지는 여러분의 브라우저에서 직접 YouTube Data API v3를 호출해 데이터를 가져와요. 키는 이 브라우저에만 저장되고 서버로 전송되지 않아요.

API 키 발급 방법 (무료)
1) Google Cloud Console에서 새 프로젝트 생성
2) “YouTube Data API v3” 사용 설정
3) 사용자 인증 정보 → API 키 만들기
4) 키 제한을 “HTTP 리퍼러”로 이 페이지 주소로 제한하면 더 안전해요

이 페이지를 파일로 직접 열면(file://) 브라우저 정책 때문에 리퍼러 제한 키가 막힐 수 있고, 영상 재생 시 “오류 153: 동영상 플레이어 구성 오류”가 뜰 수 있어요(Referer 헤더가 전송되지 않기 때문). GitHub Pages 등으로 호스팅해서(https://) 쓰는 걸 추천해요.

오늘의 표현 자동 인식
영상 설명(설명란)에 “오늘의 표현: ...” 또는 “Today's Expression: ...” 형식의 줄이 있으면 자동으로 찾아 홈 화면에 보여줘요. 없으면 스크립트 상단 EXPRESSION_OVERRIDES 에서 회차별로 직접 지정할 수 있어요.
